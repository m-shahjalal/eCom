
# HOMY -(eommerce site)


This is a full-stack eCommerce site where they sell nursery staffs

## Features

- cart
- wishlist
- payment with stripe
- profile CRU
- review section
- Order section
  
## Technology used

- **react**
- **react-redux**
- react-router
- **formik**
- express
- mongoose
- **cloudinary**
- **nodejs**
- **jsonwebtoken**
